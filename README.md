# Excalidraw

simple excaldraw example.

## Usage

```bash

# build with npm
npm run build

#fix new main.xxx.js and main.xxx.css in bprint.yaml

# export config file
export TEMPHIA_BDEV_BPRINT_CONFIG=".temphia/bprint.yaml"

# export as zip
temphia-cli  bdev zip

# upload to from admin/bprints -> new
# then instance it



```
